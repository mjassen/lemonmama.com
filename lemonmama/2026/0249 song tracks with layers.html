Pet Peeve。 Song Tracks With Layers。 Result is Sly False Dichotomy 003。




New pet peeve: songs tracks with layers. Just record it live folks, one layer with one track.
Because reality.
Not fake.

Today again heard a track, where again, the singer's own voice was overlapping itself. 
So obviously overlayed tracks.
And felt that betrayal feeling like dishonest, like it burst my bubble that this is a recording.

Maybe there's a bigger issue here.
Where I am conflating recordings, with real live singing.
If so then shame on me.
Also if so, then time to go see the doctor, or the priest. Or both.



However then, thus, got this feeling again.
Natural conclusion.
Must be false dichotomy.

How about knowing when it is layers, and also knowing when it is live recording.
And listening to both/either.
As desired.
If I like a layered track song, then do listen to it while knowing that it is a layered track song. nice.
And If I like a single track live-recorded song, while knowing it is that, then do listen to that. nice.
